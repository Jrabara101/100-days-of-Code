Core Logic Specs

  • Input Guard: if (isLocked || card.isFlipped) return; — Strictly enforced in the pointer handler.
  • Matching Algorithm:
    1. Flip Card A → Set firstCard.
    2. Flip Card B → Set secondCard, set isLocked = true.
    3. Compare cardData.id.
    4. Match: Pulse animation, keep visible, increment matchesFound, resetTurnState().
    5. Mismatch: Wait 1000ms, flip both back via tweens, resetTurnState().
  • Clean Minimalist UI:
    • Background: #F5F6FA (Light gray/white).
    • Cards: White with thin colored borders and bold sans-serif text.
    • Header: High-contrast dark text with a progress bar for the timer.
