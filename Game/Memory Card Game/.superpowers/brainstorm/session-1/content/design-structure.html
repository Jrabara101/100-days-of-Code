Design Section 1: Project Structure

A modern Vite + TypeScript setup with focused Scene logic

File Structure
memory-card-game/
├── src/
│   ├── main.ts            // Game config and initialization
│   ├── style.css          // Clean minimalist global styles
│   ├── scenes/
│   │   └── GameScene.ts   // Main game logic (Scene-Centric)
│   └── types/
│       └── index.ts       // TypeScript interfaces
├── public/                // Static assets (if needed)
├── index.html             // Entry point
├── package.json
└── tsconfig.json

High-Level Architecture

  • State Management: Uses a GameState enum (IDLE, LOCKED, MATCHED) to handle input locking.
  • Timer System: Internal Phaser Clock (TimeEvent) for the 45s countdown.
  • Grid Logic: 4x4 coordinate calculation in a single loop during create().
