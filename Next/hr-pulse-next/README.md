# HR Pulse Next Prototype

This Next.js app renders your existing HR Portal HTML prototypes as route-based screens.

## Source prototypes

The app reads from:

- `../HR Portal/hr_pulse_admin_dashboard/code.html`
- `../HR Portal/hr_pulse_candidate_profile/code.html`
- `../HR Portal/hr_pulse_employee_checklist/code.html`
- `../HR Portal/hr_pulse_onboarding_tracker/code.html`
- `../HR Portal/hr_pulse_recruitment_pipeline/code.html`
- `../HR Portal/hr_pulse_team_leave_calendar/code.html`

## Routes

- `/` Home screen with links
- `/admin/dashboard`
- `/recruitment/pipeline`
- `/recruitment/candidate-profile`
- `/onboarding/tracker`
- `/onboarding/employee-checklist`
- `/leave/team-calendar`

## Run locally

```bash
npm install
npm run dev
```
